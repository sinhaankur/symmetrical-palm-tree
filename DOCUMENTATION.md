# Documentation

## Deployment Types

### 1. SaaS (Software as a Service)
- Hosted and managed by BuildingAi-Cloud.
- Customers access the application via the cloud.
- Pricing is subscription-based and depends on usage, features, and support level.
- Each customer receives a secure, private, and isolated environment.
- Ideal for organizations seeking rapid deployment and minimal IT overhead.

### 2. On-Premise
- Deployed on the customer's own infrastructure (servers, private cloud, etc.).
- Pricing is customized based on deployment size, support, and feature requirements.
- Suitable for organizations with strict security, compliance, or customization needs.
- The application is delivered as a secure, private instance for each customer.

## Security & Privacy
- Every customer (SaaS or On-Premise) receives a logically and physically isolated environment.
- Data is never shared between customers.
- Security best practices are followed for authentication, authorization, and data protection.
- Regular security audits and compliance checks are performed.

## Pricing
- **SaaS:**
  - Tiered subscription plans (contact sales for details)
  - Add-ons and premium features available
- **On-Premise:**
  - One-time setup fee + annual maintenance/support
  - Custom pricing based on requirements
