# symmetrical-palm-tree

A modern Front End web application for facility management, supporting both SaaS and On-Premise deployments.

## Overview
This product is designed to serve two types of customers:

- **SaaS (Software as a Service):**
  - Hosted and managed by us.
  - Ideal for customers who want a hassle-free, cloud-based solution.
  - Pricing and features are tailored for multi-tenant, scalable deployments.

- **On-Premise:**
  - Deployed and managed on the customer's own infrastructure.
  - Suitable for organizations with specific security, compliance, or customization needs.
  - Pricing and software packages are customized for each customer.

## Features
- Intelligent facility management platform
- Operational excellence, security, and sustainability
- Customizable for different deployment models

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/BuildingAi-Cloud/symmetrical-palm-tree.git
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Start the development server:
   ```bash
   pnpm run dev
   ```
